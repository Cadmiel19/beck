**1.** O que é o Node.js?
Node.js é um ambiente de tempo de execução (runtime) de código aberto e multiplataforma que permite executar códigos JavaScript no lado do servidor (back-end), fora de um navegador web

**2.** O Node.js usa qual linguagem de programação?
A linguagem utilizada é o JavaScript.

**3.** O Node.js roda no navegador ou fora dele?
Ele roda fora de um navegador web.

**4.** Para que serve o Node.js?
Ele serve para executar o JavaScript fora de um navegador, principalmente no servidor backend.

**5.** Qual comando é usado para executar um arquivo Node.js no terminal?
O comando é o "node".

**6.** O que é o **npm**?
Ele é o gerenciador de pacotes padrão para o Node.js.

**7.** O que significa a palavra **módulo** no Node.js?
No Node.js, um módulo é uma unidade funcional, pequena e reutilizável de código, geralmente contida em um único arquivo JavaScript.

**8.** O Node.js é mais usado no **front-end** ou no **back-end**?
É mais utilizado no back-end.

**9.** O que é o arquivo **package.json**?
O arquivo package.json é o coração de qualquer projeto Node.js ou aplicação JavaScript que utilize npm (Node Package Manager) ou Yarn.

**10.** O Node.js é gratuito ou pago?
Ele é gratuito.
